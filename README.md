# Notizen App

Kleine Notizen App mit Frontend und Backend.

Man kann:
- Notizen speichern
- Notizen löschen
- Datum und Uhrzeit sehen

Die Notizen werden in einer JSON Datei gespeichert und bleiben nach einem Neustart erhalten.

Technologien:

- Python (Flask)
- HTML / JavaScript
- Docker
- Docker Compose

Projekt starten:

Im Projektordner eingeben:

docker compose up --build

App öffnen:

Frontend:
http://localhost:8083

Backend Test:
http://localhost:3000/test

Health Check:
http://localhost:3000/health
